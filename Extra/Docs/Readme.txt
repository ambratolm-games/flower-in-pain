1. System Requirements

================================================================================

Operating System : Microsoft® Windows® 2000 / XP / Vista
                                  (Windows Vista or XP recommended)
                                  *Note: 64-bit Editions are not supported. 
              Processor : Intel® Pentium® III 1.0GHz or higher processor.
                                   (Pentium® 4 2.0GHz or higher processor recommended) 
                  Memory : 256 MB or higher of RAM (512 MB or higher recommended) 
                   Display : 1024×768 or higher resolution 
              Hard Disk : 200 MB of available hard-disk space 

-Gamepad support:
	
	Most gamepads with dual analog sticks are supported.

2. Known Issues

================================================================================

2.1. Installing video drivers

If you experience any graphical problems, please install the recommended video
drivers for your video card. Links to the most popular video cards manufacturers 
are available at the end of this document.

--------------------------------------------------------------------------------
2.2. Installing sound drivers

If you experience any sound problems, please install the latest sound
drivers available for your sound card. Links to the most popular sound cards
manufacturers are available at the bottom of this document.

--------------------------------------------------------------------------------
2.3. Installing DirectX

To get the latest version of DirectX, please visit :
	http://www.microsoft.com/windows/directx/downloads

--------------------------------------------------------------------------------	
2.4. Installing font

If you experience any text legibility problems, please make sure that "Footlight MT Light" writing font is installed, if not you can manually install it from the “Fonts” folder you’ll find in the game folder (…\Program Files\Elhafid Games\Flower in Pain\Fonts).

--------------------------------------------------------------------------------	
2.5. Windows Service Packs

It is recommended to install the latest service packs for all operating systems.
*Note: 64-bit Editions are not supported.
